# Executive Sales Dashboard (Excel)

## Overview
This project contains an Excel-based dashboard created to analyze multi-year sales data.  
The goal was to summarize overall performance and present key insights in a way that is easy for stakeholders to understand and use for decision-making.

## Key Features
- Interactive dashboard built using Pivot Tables and slicers  
- Trend analysis across multiple years  
- Category-level sales comparison  
- Geographic sales distribution by state  

The dashboard supports filtering by year and state for focused analysis.

## Tools Used
- Microsoft Excel  
- Pivot Tables to aggregate and summarize sales data  
- Pivot Charts to highlight trends and comparisons  
- Slicers for year and state-based filtering  
- Minimal layout and formatting to improve readability  

## Files in This Repository
- `Dashboard.xlsx` – Main Excel file with the interactive dashboard  
- `Dashboard.pdf` – PDF version for quick review  
- `Dashboard.png` – Screenshot of the executive dashboard  

## Key Observations
- Sales demonstrate consistent growth over time  
- Technology is the leading revenue-generating category  
- California contributes the highest state-level sales
